# app_nsi
Un site dynamique réalisé en Java à l'aide de la framework Springboot et le template engine Thymeleaf en suivant le modèle M.V.C. pour un projet de nsi.
Les ressources (images, musiques, vidéos) utilisées dans le projet sont destinées uniquement à un usage éducatif.
Elles ne m'appartiennent en aucun cas.
